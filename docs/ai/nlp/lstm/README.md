# LSTM
## LSTM简介

长短时记忆网络（Long Short-Term Memory，LSTM）是一种特殊的循环神经网络（RNN）架构，专门设计用于解决传统RNN中存在的长期依赖问题。LSTM通过引入门控机制来控制信息的流动，从而更好地捕捉和记忆长序列中的重要信息。
