## 中式恐怖

### 概述

在传统恐怖故事中，恶人注定没有好下场，好色的人注定死在妖怪受伤，这些故事并不吓人，因为他不是冲着吓人取得。举个例子，传统的恐怖故事中，没有那种红色绣花鞋，家族世代被诅咒的剧情，因为这不符合预言和教条的内核。传统恐怖故事更多是教育人。传统故事中，苏区的人一定是因为他们没法痛改前非，犯下了在世人严重不可原谅的罪。
