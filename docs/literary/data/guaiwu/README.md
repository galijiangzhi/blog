## 烛龙

钟山之神，名曰烛阴，视为昼，瞑为夜，吹为冬，呼为夏，不饮，不食，不息，息为风。身长千里。在无䏿之东。其为物，人面，蛇身，赤色，居钟山下。《海外北经》
西北海之外，赤水之北，有章尾山。有神，人面蛇身而赤，直目正乘，其瞑乃晦，其视乃明，不食不寝不息，风雨是谒。是烛九阴，是烛龙。《大荒北经》

烛龙在山海经中的形象是人面，蛇身，红色。
