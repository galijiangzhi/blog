## 异兽

### 天狗

赤水西三百里有一峰，曰阴峰。浊浴之水出焉，而南流于番泽。其中多文贝，有兽焉，曰天狗，其状如狸而白首，其音如榴榴，可以御凶。

### 烛龙

西北海之外，赤水之北，有章尾山。有神，人面蛇身而赤，直目正乘，其瞑乃晦，其视乃明，不食不寝不息，风雨是谒。是烛九阴，是烛龙。
大荒西北之外，有幽冥无日之国，不见天光，非烛九阴不能明。
